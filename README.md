# MediSphere

MediSphere is a Healthcare Management System developed using Java Spring Boot, Angular, and MongoDB.

## Features
- Patient Management
- Doctor Management
- Vitals Management
- Consent Management
- FHIR Integration
- Kafka Integration
- Digital Twin Dashboard

## Tech Stack
- Java 17
- Spring Boot
- Angular
- MongoDB
- Apache Kafka
